# RemoteSplit

## Overview
RemoteSplit is a C++ application designed to facilitate remote desktop sharing and collaboration. It leverages the Windows Remote Desktop Protocol (RDP) at a low level to enable a user to share their desktop screen with another user, allowing for remote access, control, and viewing. The project includes a robust event handling system for remote desktop sessions and efficient data transfer between the sharer and the viewer.

## Features
- **Remote Desktop Sharing**: Share your desktop screen with another user in real-time using the Windows RDP protocol.
- **Low-Level RDP Integration**: Direct implementation of RDP for efficient and reliable remote desktop functionality.
- **Event Handling**: Custom handling of remote desktop session events for smooth operation.
- **Efficient Data Transfer**: Optimized protocols for transferring screen data between sharer and viewer.
- **Cross-Platform Support**: Built to work across different operating systems.

## How to Run
1. Clone the repository and navigate to the project directory.
2. Compile the project using a C++ compiler.
3. Run the application to start sharing or viewing a desktop.
